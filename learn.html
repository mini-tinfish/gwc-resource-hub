Explore beginner to advanced coding resources curated by GWC @ Augusta.

🌐 Web Development

Learn HTML, CSS, and JavaScript fundamentals — build your first websites and interactive apps.

  • web.dev — 🟢 Beginner | Modern tutorials on HTML, CSS, and performance best practices.
  • freeCodeCamp — 🟢 Beginner → 🔵 Advanced | Full learning paths for web development, data, and more.
  • JavaScript30 — 🟡 Intermediate | 30-day coding challenge to learn vanilla JavaScript through real projects.
  • Codecademy — 🟢 Beginner | Interactive tutorials for HTML, CSS, JS, and frameworks like React.

📊 Data Science & Analytics

Resources to develop your Python, SQL, and BI skills for data-driven projects.

  • DataCamp — 🟢 Beginner → 🔵 Advanced | Learn Python, SQL, Tableau, and Power BI with interactive notebooks.
  • Kaggle Learn — 🟡 Intermediate | Micro-courses on data visualization, machine learning, and AI tools.
  • Pandas Documentation — 🔵 Advanced | Official Python library for data analysis and cleaning.

🛡️ Cybersecurity & Networking

Practice ethical hacking, networking, and defensive security skills.

  • TryHackMe — 🟢 Beginner → 🟡 Intermediate | Guided virtual labs for learning ethical hacking.
  • Hack The Box — 🟡 Intermediate → 🔵 Advanced | Realistic penetration testing environments.
  • Coursera: Intro to Cybersecurity — 🟢 Beginner | Learn core cybersecurity principles with practical labs.

🧰 Tools & Version Control

Master Git, GitHub, and command-line tools used in real-world development.

  • Learn Git Branching — 🟢 Beginner | Interactive visual tutorials for mastering Git commands and branches.
  • Augusta Git Tutorial — 🟢 Beginner | Student-made guide for version control basics.
  • Exercism — 🟢 Beginner → 🔵 Advanced | Over 60 languages with mentor feedback and test-driven challenges.

📺 YouTube Learning Channels

Watch free, high-quality coding lessons from experienced developers.

  • Tech With Tim

    Focused on Python, web development, and computer science concepts. Tim teaches everything from beginner tutorials to AI, game development, and project-based learning.

  • mCoding

    Explores advanced Python programming, computer science theory, and software engineering best practices — all explained with clean code examples and humor.

  • Core Dumped

    Covers low-level computing, computer architecture, operating systems, and C/C++ development. A great resource for understanding how computers really work under the hood.

  • freeCodeCamp.org

    A nonprofit that publishes full-length, university-quality courses for free — from web development and data analysis to machine learning and cybersecurity.

  • Dave Gray

    Practical, beginner-friendly web development tutorials covering HTML, CSS, JavaScript, React, and backend technologies. Known for clear explanations and project-based lessons.
